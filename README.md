# carlcruzsnaps static site

## How to use

1. Replace the placeholder files in `/images` with your own photos.
2. Keep the names as:
   - photo-01.jpg
   - photo-02.jpg
   - photo-03.jpg
   - etc.
3. Open `index.html` in your browser to test.
4. Upload this folder to GitHub.
5. Enable GitHub Pages from the repository settings.

Recommended export:
- JPG
- 1600–2000px long edge
- 80–85% quality
