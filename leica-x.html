Fujifilm Recipe

Leica X

If you’re looking to get those classic Leica tones on your Fujifilm X camera, here’s a custom recipe I’ve been using.

I originally designed this for my Fujifilm X100VI to emulate the coveted Leica Colour. I started with Velvia for its punch but eventually settled on Classic Chrome for its subtlety. To balance it out, I bumped the colour all the way up until it felt just right.

This recipe was built on an X-Trans V sensor and should work well on Fujifilm cameras with the same sensor, such as the X-T5. If you’re using an older model, try punching in as many of the settings as possible and tweak from there.

Settings:

  • Film Simulation: Classic Chrome
  • Grain: Off
  • Chrome Effect: Strong
  • FX Blue: Strong
  • White Balance: Auto (R: +1, B: -2)
  • Dynamic Range: Auto
  • DR Priority: Off
  • Tone Curve: Highlights -1, Shadows 0
  • Color: +4
  • Sharpness: 0
  • High ISO NR: -4
  • Clarity: 0
